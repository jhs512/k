# 장희성 프로필 및 선호 (사용자 직접 구술, 2026-06-10)

나는 장희성이고 86년생 한국 남자이고, IT 강사이면서, 풋살을 좋아하고, 한식을 좋아해. 대중문화에 관심이 없고, 돈 버는 거 좋아하고, 그렇다고 인색하지는 않고, 인과율을 믿어서 뿌린 대로 거둔다고 믿고 있어.

스프링부트 강의를 주로 하고, 스프링부트 세팅 시에는:
- yaml 방식
- 최신 안정 버전
- JDK 25 이상
- Group: com, Artifact: back
- Gradle은 KTS
- 필수 의존성: Spring Web, 시큐리티, H2, JPA, DevTools, (자바+스프링이면 롬복)
- 시큐리티 기본 설정은 모두 허용, H2 콘솔 작동되어야 하고, CSRF 다 끄기
- 타임리프 방식보다는 REST 선호
- 개발 모드(application-dev.yml)는 H2 파일 모드
- 테스트 모드(application-test.yml)에서는 H2 메모리 모드
- 테스트는 메모리 DB에서 실제 HTTP 콜 요청 날린 후 트랜잭션 롤백으로 초기화하는 방식을 선호
- 모든 SQL은 자세히(예쁘게) 출력하는 걸 선호, binding(바인딩 파라미터)이란 extract까지
